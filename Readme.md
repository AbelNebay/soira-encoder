# SoiraEncoder

## Overview
SoiraEncoder is a desktop media processing application that converts videos into a secure Soira format and encrypts them for controlled distribution.

## Features
- Video conversion to Soira format
- Media encryption for content protection
- Batch processing support
- Optimized output for secure playback systems

## Tech Stack
- Java (NetBeans)
- FFmpeg integration
- Custom encryption algorithms

## Use Case
- Secure video distribution
- Educational content protection
- Offline media delivery systems
